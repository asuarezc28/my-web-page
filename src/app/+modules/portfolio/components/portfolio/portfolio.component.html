MY WEB

This application has been developed with Angular 10, Flex-Layout, Material Angular and Bootstrap.

GROWTH VIEWER

This application has been developed with the constructor of applications "Webappbuilder" of ESRI. A new design has been made and a new widget has been developed, "Footprint Growth Chart" where there are two selectors to choose the period of years in which we want to see the city growth. In this consult , It will show in the map only the the corresponding polygons between the start year and the end year. In the other hand, a graphic will be generated with the library "Chart.js" and a table with information corresponding to this growth in hectares and percentages. Finally, a new functionality has been added to the widget "Query" where making a query by geometry, the report generated by the widget in PDF also shows a graph with said query.

MOBILITY VIEWER

This application have been developed with the constructor ESRI's “Webappbuilder” application. On this occasion, developed the widget "Open" where in the main menu allows us to access different widgets housed in the application. Within each of these widgets we can select an entity of a listing and the map will zoom to the selected element.

3D VIEWER

This application have been developed with the constructor ESRI's “Webappbuilder” application. It show the flats of Panama City in 3D, differentiating the category of each one of them by colors.

PERMISION VIEWER

A widget has been developed in this viewer with the aim of know if in a certain area within the layer of "Zoning" we could build taking into account the use of soil that we propose in the selector, as well as the number of plants to be built. The widget will approve said construction as long as the permitted use selected be the same as the zoning layer and the number of plants is equal or less, then the widget will show us a message that it can be built and will also enable us to form to fill in the fields proposed for said construction and the resulting new graph will be added to the service of the "Construction Permits" layer with attributes filled in the form.

WALKING TRAILS

Application about the trails of the island of La Palma, developed with React.js and ESRI API 4.12. All the trails on the island are represented on the map differentiated by its difficulty. In the right side menu, the user can choose what trails will be shown on the map and you can select a certain difficulty. Once the user selects the difficulty and click clicking on the "Consult" button will generate in the same menu a list of the trails that have such difficulty, which the user can easily identify by clicking on the icon of a magnifying glass located to the right of the trail name, answering the map with a zoom to that trail. Finally the user can also search a certain trail from the list with a search engine typing the name on the top of this listing
